# AI Critical Audit Criteria (AI 검수 헌장)

이 문서는 AI Agent가 시스템의 데이터 품질을 검수할 때 준수해야 할 엄격한 기준입니다.

## 1. 연령 적합성 (Age Precision) - 가중치 40%
- **초등 방어:** 제목/설명에 '초등', '교과', '학년'이 언급되었으나 `target_months_min` < 84(7세)인 경우: **즉시 불합격 및 -50점.**
- **영아 방어:** '보드북', '헝겊', '초점' 책이 `target_months_min` > 24인 경우: **부적합.**
- **물성 일치:** '양장본'이면서 텍스트가 500자 이상인데 `target_months_min` < 48인 경우: **부적합.**

## 2. 영역 분류 논리 (Area Logic) - 가중치 30%
- **키워드 모순:** '수학' 키워드가 있는데 `자연탐구`가 없는 경우.
- **분류 부재:** 설명은 풍부한데 `developmental_areas`가 빈 배열(`[]`)인 경우: **치명적 결함.**
- **맥락 오독:** '구름'이 기상 현상이 아닌 상상력 소재(예: 구름빵)인데 `자연탐구`로 분류된 경우.

## 3. 개인화 데이터 (Personalization) - 가중치 20%
- **에너지 레벨:** 활동적인 놀이책인데 `energy_level` < 7인 경우.
- **워크북 감지:** '스티커북'인데 `is_workbook`이 false인 경우.

## 4. 데이터 완성도 (Data Quality) - 가중치 10%
- **이미지:** `cover_url`이 `cover200`(저화질)인 경우.
- **코멘트:** AI 코멘트가 지나치게 짧거나("재미있어요" 등) 앞뒤 문맥이 안 맞는 경우.

## [최종 점수 산정 방식]
- 모든 항목이 완벽할 때만 100점.
- 사용자에게 잘못된 추천(초등책을 아기에게 등)을 할 가능성이 보이면 60점 이하 낙제 부여.
